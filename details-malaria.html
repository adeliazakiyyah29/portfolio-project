Identifying Malaria with Deep Learning

  • Problem Definition

    • Malaria is a life-threatening disease caused by Plasmodium parasites, transmitted by Anopheles mosquitoes. According to the WHO, over 228 million cases are reported annually worldwide.

    How can a blood test detect malaria? Traditionally, malaria diagnosis requires manual examination of blood smear slides under a microscope by trained experts, which is time-consuming and error-prone. This project proposes a deep learning-based image classification model to automatically detect malaria parasites in red blood cell images.

  • Data Collection
    • Dataset: 2,000 RGB images of blood smears labeled as infected and uninfected
    • Source: National Institute of Health (NIH) public dataset
  • Exploratory Data Analysis (EDA)
    • Parasite nuclei appear as dark purple dots in infected cells
    • Image sizes vary; some duplication and class imbalance present
    • Images collected from multiple microscopes, contributing to noise
    • Class distribution was slightly skewed
  • Data Preprocessing
    • Image resizing to 128x128 pixels
    • Augmentasi Citra (using random rotation, flip, and zoom)
    • Split dataset :
      • Train: 80%
      • Validation: 20%
      • Test: 10 new unseen images
  • Modeling
    • Model: Convolutional Neural Network (CNN) with reduced layers based on AlexNet
    • Loss Function: categorical crossentropy
    • Optimizer: Adam
    • Epochs: 50
    • Metriks: Accuracy, confusion matrix
  • Evaluation and Result
    • Achieved 98% test accuracy in identifying whether a cell is infected with malaria
    • Model successfully learned parasite pattern features in blood cell images
